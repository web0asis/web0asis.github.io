Stock Screener

System requirements

  • PHP 5.4.x or above
  • cURL enabled
  • [PHP setting] short_open_tag = on

Setup

Unzip the package contents and copy all files to your web server via FTP or control management panel (cPanel, Plesk etc). That's it, no more configuration or database installation is required.
