About

This password generator is a completely free online password generating tool. The highly secure passwords that you create are NOT stored anywhere. The passwords are generated clientside in your browser only using javascript. No information is collected using this generator.

Advice

  1. We suggest using passwords that are 15 characters or longer.
  2. Never use words in the dictionary as any part of your password.
  3. Make sure your email is safe and secure with an incredibly strong and unique password.
  4. Don't use truthful answers that people can guess or Google for the forgot your password reset questions.
  5. Databreaches happen nearly every day now. Every website you visit should have its own unique password.
  6. We suggest using password manager software to help you keep track of your unique passwords.
