Keyboard Shortcuts in the Notepad

Shortcuts

Here is the complete list of key combinations that will help you to speed things up when using the Notepad.

Action PC Mac
Open file Ctrl + O Command + O
Save Ctrl + S Command + S
Download file Ctrl + Shift + S Command + Shift + S
Print Ctrl + P Command + P
Undo Ctrl + Z Command + Z
Redo Ctrl + Y Command + Y
Cut selected text Ctrl + X Command + X
Copy selected text Ctrl + C Command + C
Highlight all Ctrl + A Command + A
Find and replace Ctrl + Shift + R Command + Shift + R
Insert date & time Ctrl + Shift + D Command + Shift + D
Open charmap Ctrl + Shift + C Command + Shift + C
Open emoji list Ctrl + Shift + E Command + Shift + E
Open font preference Ctrl + Shift + G Command + Shift + G
Toggle fullscreen Ctrl + Shift + F Command + Shift + F

Note: Most web browsers have built-in additional shortcuts. You may need to click within the editor context to activate the shortcuts listed above.
